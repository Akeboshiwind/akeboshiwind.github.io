A draft: Shopping Sorter

2026-04-10 · updated 2026-04-12 · history

I like to keep my shopping list ordered by how I walk through the shop (I walk in the same order each time more or less). Usually I would do this manually on the walk to the shop, but after a year that got tedious. So I asked Claude to build a script for Drafts that will do this for me automatically!

The way it works is:

  1. In a new draft I’ll make a list of things I want to add: milk, bread, tomatoes
  2. I trigger the script
  3. For unknown items I select the aisle they belong in or create a new aisle
  4. If an aisle is created I select which aisle it goes before/after

That produces two notes:

  1. An archived json file with the list of aisles and learned item -> aisle mappings
  2. A shopping list

Some additional features:

  • It supports bullet point or comma separated input drafts
  • Capitals, amounts (<item> x2) etc get mapped to the correct aisle
  • Completed (- [x] <item>) items are deleted
  • Input drafts are archived
  • You can re-sort the shopping list by running it on it directly
