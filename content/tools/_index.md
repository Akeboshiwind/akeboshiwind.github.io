A collection of useful tools mostly created with Claude.

## Available Tools

- [Bitemporal Visualizer](/tools/bitemporal/) - An interactive visualizer for bitemporal data that tracks two time dimensions: valid time and system time.
- [Bill Splitter](/tools/bill/) - Split restaurant bills among friends with custom percentages and currency conversion.
- [YNAB Reimbursement](/tools/ynab-reimbursement/) - Track reimbursement transactions for YNAB.
