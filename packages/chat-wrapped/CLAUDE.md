# Chat Wrapped

Telegram chat analytics visualization - "Spotify Wrapped" style stats for group chats.

## Commands

```bash
bun install       # Install dependencies
bb build          # Build for local dev
bb build:ci       # Build for production (with base-path)
bb dev            # Dev mode with watch + serve
bb serve          # Serve built files
bb clean          # Clean build artifacts
```

## Tech Stack

- **Squint** - ClojureScript to JavaScript compiler
- **Bun** - JavaScript runtime and bundler
- **Babashka** - Task runner
- **Tailwind CSS** - Styling
- **React 19** - UI (via Squint JSX)

## Structure

```
src/
  index.html       # HTML template
  assets/          # Images, profile photos, etc.
  talk/
    app.cljs       # Main entry point
    *.cljs         # Components
build/             # Compiled JS (generated by squint)
target/public/     # Final build output
```
