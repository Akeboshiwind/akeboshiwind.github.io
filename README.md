# Blog

A blog built on Jekyll

## Getting started on the project

First clone the project then run the following command to properly initialize everything:

```bash
./scripts/init
```

You can then use the project normally, use the following command to serve the blog locally for testing:

```bash
bundle exec jekyll serve --drafts
```

## Releasing

After you've committed your changes to the master branch you can run the following command to release the changes to the live site:

```bash
./scripts/release
```
