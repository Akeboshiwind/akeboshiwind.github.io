# Blog by the Rocks

Personal blog built with [Astro](https://astro.build/).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview  # Preview the build
```

## Structure

- `src/content/blog/` - Blog posts (Markdown/MDX)
- `src/pages/` - Astro pages
- `packages/` - Interactive tools (Squint/ClojureScript SPAs)
